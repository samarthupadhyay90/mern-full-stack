🎯 Guessing Number Game - How to Play?

Welcome to the Guessing Number Game! 🕹️ Test your luck and guessing skills by finding the correct number.

🎮 Game Rules:

1️⃣ Enter the maximum number (sets the range).

2️⃣ The computer will generate a random number between 1 and your chosen max.

3️⃣ Try to guess the number!

4️⃣ You will get a hint after each guess:

- "Your guess was too small" → Try a higher number.

- "Your guess was too large" → Try a lower number.

5️⃣ Keep guessing until you find the correct number! 🎉

⏹️ How to Exit the Game?

If you want to quit, type "quit" instead of a number.

🏆 Winning the Game:

If your guess is correct, you win! 🎊

The game will display: "You are right, congratulations! 🎉 The random number was [X]"

⚠ Important Notes:

✔️ Enter only numbers (except for "quit").

✔️ The game won’t accept letters or symbols.

✔️ Have fun and test your skills! 🎮
